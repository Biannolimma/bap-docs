# Como contribuir

Esta documentação descreve como qualquer pessoa pode contribuir para o projeto BlockAndPlay (BAP), sem revelar processos internos sensíveis.

## Como sugerir melhorias?
- Crie uma issue pública no repositório principal ([blockandplay](https://github.com/Biannolimma/blockandplay)).
- Para dúvidas genéricas, use os canais públicos informados no README.

## Fluxo básico de contribuição
1. Faça um fork do projeto
2. Crie uma branch para sua contribuição
3. Abra um Pull Request no repositório principal

Ajude a tornar o projeto melhor! Não exponha informações privadas ou internas em contribuições públicas.
