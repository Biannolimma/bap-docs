# Instalação

A documentação pública de instalação do BlockAndPlay (BAP).

## Pré-requisitos
- Git instalado
- Python (>=3.8) ou Node.js, conforme a versão do BAP
- Acesso ao repositório blockandplay

## Passos

```bash
git clone https://github.com/Biannolimma/blockandplay.git
cd blockandplay
```

Siga as instruções do README do projeto para executar o setup. A documentação completa e exemplos estão disponíveis no projeto original.
