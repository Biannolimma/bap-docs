# APIs & Funcionalidades

Esta página lista os principais módulos e funcionalidades públicos do BlockAndPlay (BAP).

## Endpoints e Módulos

Consulte os arquivos de código do repositório principal para detalhes técnicos. Não há endpoints sensíveis exibidos nesta documentação pública.

- **Gerenciamento de Blocos**: Criação, edição e exclusão de blocos
- **Visualização de Atividades**: Interfaces para acesso aos dados
- **Autenticação básica**: Login de usuários

Para APIs REST ou outros detalhes públicos, consulte o README e a referência no repositório principal.
