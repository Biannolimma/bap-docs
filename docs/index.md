# BlockAndPlay (BAP)

Bem-vindo à documentação pública do BlockAndPlay (BAP).

Este projeto foi criado para facilitar a organização e gerenciamento digital de blocos e atividades para diferentes usuários.

## Recursos
- Blocos personalizados
- Organização e visualização de atividades
- Interfaces responsivas
- Extensível conforme futuras necessidades

Para instruções de instalação, consulte [Instalação](instalacao.md). Para APIs e integrações, veja [APIs & Funcionalidades](api.md). Para contribuir, acesse [Como contribuir](contribuicao.md).
